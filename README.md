# Users Management Project

A simple web application for managing users, built with HTML, CSS, JavaScript, and FastAPI. The application allows users to:

- Add new users to the database.
- Update existing user details.
- Delete users from the database.
- View all user data in a dynamic HTML table.

## Features

- **Dynamic Table:** Displays user data fetched from the API.
- **Add Users:** Add new user entries via a form.
- **Update Users:** Populate user details in a form for editing and save updates.
- **Delete Users:** Delete specific user entries with confirmation prompts.
- **API Integration:** Communicates with a FastAPI backend for CRUD operations.

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** FastAPI
- **Database:** PostgreSQL
